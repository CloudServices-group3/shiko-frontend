# Shiko LMS - Frontend 🎓

The frontend application for the Shiko Learning Management System. 

🚀 **Live Demo:** [https://shiko-frontend-silk.vercel.app](https://shiko-frontend-silk.vercel.app)

---

## 🛠️ Tech Stack
* **Framework:** Next.js (React)
* **Styling:** Tailwind CSS
* **Communication:** Fetch for API integration with our Azure-based microservices
